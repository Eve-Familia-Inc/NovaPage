基本的な使い方

Pythonスクリプト内でNovaを利用する方法を学びます

まずはインポートしましょう

Novaが他のパッケージと競合している場合は こちらをお読みください 

import Nova

とりあえずHTTPサーバーを動かす

IPアドレス：127.0.0.1 ポート：7777
root_path引数に指定した/var/www/html以下のファイルからMIMEタイプに一致するものだけ自動的にメモリに格納。
Nova.Service.Start()でサーバーが起動し、[CTR] + Cでサーバーを終了することができます。

import Nova

Nova.Service(
    host="127.0.0.1",
    port=7777,
    root_path="/var/www/html"
).Start()

任意のブラウザでhttp://127.0.0.1:7777にアクセスすると、404 Not Foundもしくは/var/www/html/index.htmlの内容が表示されます。
