はじめる

基本的なセットアップ方法と、Novaパッケージの概要を知りましょう

とりあえずインストール！

Pythonのバージョンは3.7以上である必要があります。
環境によってはpipがPython2系で動作する可能性があります。その場合はpip3を利用しましょう。
Novaが他のパッケージと競合している場合は こちらをお読みください 

pip install Nova-EveFamilia

ソースの構成

Nova/
    __init__.py
    Core/
        __init__.py
        ServerBase.py
        Stream.py
    Server/
        __init__.py
        Http.py
        Defines.py
ファイル 説明
Nova/__init__.py import Novaで読み込まれるソースです
Nova/Core/__init__.py パッケージ構築用の空のファイル
Nova/Core/ServerBase.py サーバーを起動しクライアントの接続を受け付け、ハンドラに引き継ぐクラスが含まれています
Nova/Core/Stream.py asyncio.start_server()をラップするAsyncStream()クラスが含まれています
Nova/Server/__init__.py パッケージ構築用の空のファイル
Nova/Server/Http.py Httpソケット通信に対する具体的な処理を行うクラスが含まれています
Nova/Server/Defines.py レスポンスコード、MIMEタイプの定数が含まれています

Githubからダウンロードしてポータブルに使う

git clone https://github.com/Eve-Familia-Inc/Nova.git
cd Nova
# ディレクトリ内でNovaが利用できるようになります。
